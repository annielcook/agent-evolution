Step 1: From Prompt to Small Agent

When people first start using large language models like ChatGPT, they often treat it like a smart search engine: one question in, one answer out.

But with a carefully designed prompt, you can do much more: You can make the model think through multiple steps, use judgment, and simulate an iterative agentic process — all in a single chat.

Instead of just asking a question, you're instructing the model to generate ideas, propose experiments, and refine solutions. This is the first step toward designing agents: Getting an LLM to act with a process, not just a reply.

Common One-Off Prompt

How can I improve recycling rates in my city?

Structured Multi-Step Prompt

You are a Civic Innovation AI Agent designed to help public entrepreneurs explore and test ideas...
